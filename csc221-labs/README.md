# CSC221 Labs

This folder is for CSC221 lab assignments.

## Layout

Put each lab in its own folder:

```text
lab-01/
lab-02/
lab-03/
```

Recommended per-lab files:

```text
lab-01/
  README.md
  main.py
  notes.md
```

Keep assignment instructions, code, and any small test data with the lab they belong to.
