# Python Lab

Python practice, scripts, tools, and project experiments.

## CSC221 Labs

Course lab work lives in [`csc221-labs/`](csc221-labs/).

Use one folder per assignment:

```text
csc221-labs/
  lab-01/
  lab-02/
  lab-03/
```

Each lab folder can hold the Python files, notes, screenshots, or data files needed for that assignment.
